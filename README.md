# 📖 [NETWORK SCRIPT PROJECT]<a name="about-project"></a>

**[formulario-webpack** Lo que tarta el proyecto es un dezspliegue en AZURE de un formulario web que se puede utilizar para recopilar información de los usuarios. El formulario está construido utilizando Webpack y se puede personalizar fácilmente para adaptarse a las necesidades específicas de cada proyecto.**

## 💻 Getting Started <a name="getting-started"></a>


Para configurar una copia local, siga estos pasos:

### Prerequsites 

Las herramientas que se neceita para poder ejecutar el proyecto son:

- [VS Code]
- [Git and GitHub]
- [ShellScript ]
- [Node.js]
- [NPM]
- [Webpack]



Clonar este  respository  en la carpeta que deseas:

```sh
cd FORM-LINTERS
git clone https://github.com/jhonsito13/FORM-LINTERS.git
```
### Install

Se instalo los paquetes necesarios para la ejecucion del proyecto:

```
1. npm install
2. npm run build
3. npm start
4. npm test
5.npm install --save-dev webpack webpack-cli
6.npm install --save-dev webpack-dev-server
7.npm install --save-dev html-webpack-plugin
8.npm install --save-dev css-loader
9.npm install -g azure-cli




```


### Usage 


Para ejecutar el proyecto, ejecute el siguiente comando:

```
./networkScript.sh
```

### Run Test

Para ejecutar la prueba, ejecute el siguiente comando o endpoint:

